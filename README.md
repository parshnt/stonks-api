# stonks-api

- wrapper around Yahoo Finance API.
- Build using CF workers.

## Endpoints


Hit the following endpoint:

https://stonks.parshnt.workers.dev/ <INSTUMENT_SYMBOL> . <MARKET_ID>

## Examples

Infosys: NSE - https://stonks.parshnt.workers.dev/INFY.NS 
RIL: BSE - https://stonks.parshnt.workers.dev/RELIANCE.BO

* NS = NSE
* BO = BSE
